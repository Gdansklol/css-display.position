## position:relative och fixed

```css
.parent {
  background: lightblue;
  position: relative;
  height: 200vh;
}

.child-one {
  background: green;
  position: fixed;
  top: 0;
  right: 0;
}

.child-two {
  background: orange;
}

.child-three {
  background: pink;
}
```
